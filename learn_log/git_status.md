# git status 指令

## 参考资料：
* [git status官方API](https://git-scm.com/docs/git-status)

## 实践总结：
* git status 指令可以带很多参数例如：指定查看忽略文件、未跟踪文件等（具体查看参考资料）， 但常用的还是默认查看完整状态信息；

## git status 常用指令：
* git status
	* 显示代码完整状态信息（相当于`git status --long`）

* git status -s
	* 显示代码简短状态信息(只显示变化的文件)
